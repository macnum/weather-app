# Installation

1. Clone the repository:

```bash
git clone https://github.com/macnum/weather-app

cd  weather-app
```

2. Install dependencies

```bash
npm install

# or

yarn install
```

3. Start the development server:

```bash
npm run dev
```

## usage

1. Open the app in your browser at `http://localhost:3000`
2. Enter a city name in the search box
3. Press Enter or click the search button
4. View current weather conditions and forecast

## API

This app uses [visualcrossing API](https://www.visualcrossing.com/weather-api/) (or specify your API)

- Get your free API key at: https://www.visualcrossing.com/weather-api/
